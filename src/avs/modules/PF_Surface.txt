NAME
	PF_Surface

SUMMARY
	Name	PF_Surface

	Type	Mapper

	Inputs	X_inter_by_yz 3D 3-space 2-vector rectilinear float, OPTIONAL
                Y_inter_by_zx 3D 3-space 2-vector rectilinear float, REQUIRED
                Z_inter_by_xy 3D 3-space 2-vector rectilinear float, REQUIRED

	Outputs	Surface_out  geom

	Parameters  Grid_X_start integer
                    Grid_X_fin integer
                    Grid_Y_start integer
                    Grid_Y_fin integer
                    Solid_num integer
                    Show_Top choice
                    Show_Bottom choice
                    Show_Sides choice
DESCRIPTION

Uses intersection data to generate surface.
Solid index picked via parameter.
Top, bottom and sides can be turned on and off.
