ParFlow Modules

NAME
     integer to file - convert an integer to a file name string

SUMMARY
     Name          integer to file

     Availability  ParFlow module library

     Type          filter

     Inputs        integer

     Outputs       string

     Parameters    Name          Type       Default       Min       Max
                   File Prefix   string     NULL                        
                   Format        string     %s%04d.ext                  

DESCRIPTION
     This module converts an integer value to a file name string.
     The file is constructed from the "File Prefix" parameter, the
     input integer, and the "Format" parameter.  For example, if we
     have:

           integer input = 12
           "File Prefix" = /home/bacchus0/falgout/prefix_
           "Format"      = %s%04d.ext

     the resulting filename is

           "File" = /home/bacchus0/falgout/prefix_0012.ext

     The browser widget in AVS is a bit clumsy, resulting in an easy-
     to-use, but non-intuitive interface for setting the "File Prefix"
     string.  To set, first use the browser to select the directory
     where the file is (to be) located.  In the example above, we would
     select the directory `/home/bacchus0/falgout'.  Now, use the
     "New File" button to define the remainder of the "File Prefix".
     In the example above, we would type `prefix_'.  Finally, type in
     a format string as above.  The `%s' matches the "File Prefix"
     string.  The `%04d' matches the integer input.

INPUTS

     integer (required)
          An integer value that matches the digit conversion specifier
          in the "Format" parameter below.

PARAMETERS

     File Prefix
          A string that matches the string conversion specifier in the
          "Format" parameter below.  This parameter usually represents
          the first part of the file name, however, this is not required.

     Format
          A standard C format string containing one string conversion
          specifier, one digit conversion specifier, and any other
          non-conversion character sequences.  Note, the string conversion
          specifier must come before the digit conversion specifier.

OUTPUTS

     File (string)
          A file name constructed from the integer input and the
          "File Prefix" and "Format" parameters.
