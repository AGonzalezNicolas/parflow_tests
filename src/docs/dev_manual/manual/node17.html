4. ParFlow Modules

There are several reasons for writing modules:

  • to provide a number of choices for a particular call-point in the code's execution graph,

  • to store "static" data that can be set up once at some initialization stage to help optimize the execution of a function,

  • to help minimize memory usage by using 's temporary data capability.
