10.3.4 Backing up an Archive

Suppose you wanted to backup to tape an archived simulation, with the run-name ``xyzzy''. Do the following :

  1. As user parflow :
    1. Get and mount a tape on an appropriate machine.
    2. Go to the directory parflow/data.
    3. Issue the command tar -chvf TAPE_DEVICE_NAME ./*/xyzzy, where ``TAPE_DEVICE_NAME'' is the name of the tape drive on the particular system.
  2. As the user performing the backup :
    1. Make an entry in the system tape log book about the tape.
    2. Checkout (and lock) the file INFO.html.
    3. Make a note that you backed up the data at this day and time in the file INFO.html.
    4. Checkin the file INFO.html.
