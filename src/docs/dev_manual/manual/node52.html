10.3.5 Removing an Archive

Before removing any data from parflow/data be sure to check the status of the data. If the data has not been backed up, do so using the back up procedure, § 10.3.4, before deleting it.

Suppose you wanted to remove an archived simulation, with the run-name ``xyzzy''. Do the following :

  1. As the user removing the archive :
    1. Check the ``Simulation Info'' link to make sure ``xyzzy'' has been backed up - if not do so now, § 10.3.4.
  2. As user parflow :
    1. Go to the directory parflow/data/output.
    2. Remove the the contents of directory ./xyzzy.
  3. As the user removing the archive :
    1. Checkout (and lock) the file INFO.html.
    2. Make a note that you removed the data at this day and time in the file INFO.html.
    3. Checkin the file INFO.html.

Normally the input data is left in place since it requires only a small amount of space.
