10.3.6 Restoring an Archive

Suppose you wanted to restore a previously backed up and removed archive, with the run-name ``xyzzy''. Do the following :

  1. As the user restoring the archive :
    1. Use the ``Simulation Info'' link and the system tape log book to figure out what tape contains the ``xyzzy'' data.
    2. Get and mount the tape on an appropriate machine.
  2. As user parflow :
    1. Go to the directory parflow/data.
    2. Issue the command tar -xpvf TAPE_DEVICE_NAME ./*/xyzzy, where ``TAPE_DEVICE_NAME'' is the name of the tape drive on the particular system.
  3. As the user restoring the archive :
    1. Checkout (and lock) the file INFO.html.
    2. Make a note that you have retrieved the data at this day and time in the file INFO.html.
    3. Checkin the file INFO.html.
