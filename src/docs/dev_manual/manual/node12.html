3.1.2 Providing Online Help for GUIs

A mechanism has been developed to provide online help for GUIs. The command

must first be added to manual sections that are to be used as online help pages (see file files.tex of the User's Manual for examples). This command produces an anchor in the online html document and nothing in the printed version. The anchor name is created by prepending the string PFAnchor to the tag argument of the pfaddanchor command.

After the online manual is created, the command do_online should be executed by the manual Makefile. This routine is located in the $PARFLOW_SRC/docs/lib directory. It parses the online manuals for the string name=PFAnchor, then creates the file pfanchors which contains a table of anchor names and the files in which they appear (note, the file is sorted by anchor name). In this way, GUIs need only search the pfanchors files to determine the location of appropriate help information.
