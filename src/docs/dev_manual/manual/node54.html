10.3.7 Using an Archive

In order to not corrupt data in the archive it is suggested that users only work with copies of the data rather than any data in  parflow/data.

Suppose you wanted to use data from an archive, with the run-name ``xyzzy''. Do the following (as any user) :

  1. Check the ``Simulation Info'' link to find the status of ``xyzzy''.
  2. If the data has not been removed, you can simply use it.
  3. If the data has been removed; you can either restore the data in the archive (§ 10.3.6) or use the README file to checkout the applicable version of the code and the input files to rerun the simulation.
