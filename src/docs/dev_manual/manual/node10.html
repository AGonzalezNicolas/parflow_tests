3.1 Writing Manuals

The manuals are written in LaTeX and converted to HTML (via the package LaTeX2HTML) to be used online. The source for these manuals is located in subdirectories of the directory $PARFLOW_SRC/docs. Various style files used to build the manuals are located in the directory $PARFLOW_SRC/docs/lib. Files ending in .sty are used in LaTeX to create printed versions of the manuals, and files ending in .perl (these files are perl scripts) are used by LaTeX2HTML to create online versions of the manuals. The files html.sty and html.perl define various commands that are part of the LaTeX2HTML package. With these commands, one can do things such as add hyperlinks to the online document which are ignored by LaTeX when creating the printed document. The files parflow.sty and parflow.perl define various commands and environments to be used by developers when writing documentation. They are discussed more fully in § 3.1.1.
