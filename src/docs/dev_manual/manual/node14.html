3.3 Writing PFIX Files

The PFIX web pages are kept under RCS control in the  parflow/html directory. The public directory is for information that is for local users (helios only servers local machines). private is used for internal use only documents for developers. Items which are to be available to everyone should go on the home page.

If you need to modify one of these files use the following procedure:

  1. Go to the directory parflow/html.
  2. Checkout (and lock) the HTML file. If the file is already writable it is probably being updated by someone else!
  3. Make the changes you want
  4. Test the new page using the PFIX link
  5. Checkin the HTML file.
