10.3 Data Archiving

It is necessary to maintain archives of important simulations generated with - where the output results are used in papers, on-line images and mpeg movies, or VHS videos. A mechanism exists and procedures have been created to facilitate easy archiving of this important data.

The mechanism for archiving data consists of :

  • A file system organization for storing the data § 10.3.1.
  • An html file for recording information about the data § 10.3.2.

The procedures for archiving data are :

  • Creating a data archive § 10.3.3.
  • Backing up a data archive § 10.3.4.
  • Removing a data archive § 10.3.5.
  • Restoring a data archive § 10.3.6.
  • Using a data archive § 10.3.7.
