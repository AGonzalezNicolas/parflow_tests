4.4 Adding ParFlow Modules

To incorporate the new module you have written into the code you need to add the source and header files to the Makefile in the parflow/src directory. The PF_SRC variable in the Makefile controls which files are compiled into the executable. Header files should go in the PF_DEPEND Makefile variable.
