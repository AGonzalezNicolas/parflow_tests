3.2 Writing HTML Files

HTML files that are created ``by hand'' (i.e. not generated by a package such as LaTeX2HTML), are kept under resource control in the directory $PARFLOW_SRC/docs/html. This directory mimics the $PARFLOW_HELP directory. To modify and install an HTML file, do the following:

  1. Check out and lock the file to be modified.

  2. Make modifications.

  3. In directory $PARFLOW_SRC/docs/html, type

    This will copy the HTML files in the current directory tree to the appropriate place in the $PARFLOW_HELP directory.

  4. Finally, use xpfci to check in the modified source.

Note: if you are creating new HTML files or deleting old HTML files, you will need to modify the above build script.
