2.5 Installing a new version of ParFlow

This section discusses how to install a new version of for general users; this is refered to by the team as the ``installed'' version of .

This will describe the steps need to install the version on the CASC machines (which currently consist of the Sparc Cluster and the SGI Onyx).

The first thing to do is install on the Sparc Cluster:

  1. Log in to the parflow account on the Sparc Sun Cluster
  2. Execute the script to build and install parflow. This will take a rather long time.

  3. Execute the test program to make sure things are running OK

  4. ParFlow should run on the default_single problem

If that worked, install on the SGI:

  1. Log into nyx (as the parflow user)

  2. Build and install parflow

  3. Execute the test program to make sure things are running OK

  4. ParFlow should run

Things should be good to go. Email ``pfusers'' to tell everyone that you have installed a new version of parflow.
