9.2.2.2 Compiling AVS/Express Modules

The compilation process in AVS/Express compiles all source code specified in the user library, so when you compile you are in effect compiling the entire user library. To compile from within Express, click on the library version of a module with source code, and select Compile from the Project pulldown menu.
